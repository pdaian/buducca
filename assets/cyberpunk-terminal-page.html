buducca://overview

Modular messaging AI framework for building autonomous chat assistants across Telegram, Signal, and WhatsApp with pluggable skills and collectors.

network-links

  • GitHub: https://github.com/pdaian/buducca
  • Telegram: https://t.me/buducca

use-cases

  • Automate daily summaries from Gmail, Slack, and calendar collectors.
  • Run a personal ops assistant via Telegram with command-based skills.
  • Create customer-support copilots that query workspace knowledge.
  • Trigger workflows from chat events and summarize outcomes instantly.
  • Prototype multi-channel AI agents with shared logic and adapters.

quickstart

python run_bot.py --config config.json

Initialize your config, connect provider keys, then run the bot command above.
